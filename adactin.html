Case Study: Adactin Hotel App Automation Testing

This project focused on automating end-to-end functional testing for Adactin Hotel Booking application, ensuring seamless user experience and robust backend operations.

Client: Adactin Hotel App (Training Application)

Industry: Hospitality / Training

Website: adactinhotelapp.com

Project Requirements

  • Automate hotel search, booking, and cancellation scenarios
  • Validate form input fields and data constraints
  • Implement a data-driven framework for multiple test data sets
  • Generate reports and capture screenshots on failures

Project Overview

The Adactin Hotel App is widely used as a training application for aspiring automation testers. The core objective was to validate critical workflows such as searching hotels, selecting rooms, booking, payment processing, and profile management. Thorough functional and UI testing was essential to ensure the app could handle a variety of user inputs and business scenarios reliably.

Given its purpose as a learning platform, maintaining a stable, defect-free environment was crucial so learners could focus on building their automation skills without encountering application issues. The project combined manual exploratory testing with an extensive automation suite to achieve these quality goals.

The Challenge

Several challenges emerged during this project:

  • Dynamic UI Elements: The application frequently updated UI layouts and introduced dynamic identifiers, making element locators highly volatile and prone to breaking.
  • Large Test Data Combinations: Validating business logic for various combinations of cities, hotel names, room types, and pricing required a scalable, data-driven approach to avoid duplicating test scripts.
  • Cross-browser Compatibility: Although designed for training, the application needed to be verified across multiple browsers to ensure consistent behavior for different users.
  • Efficient Reporting: Given its educational purpose, clear, professional reports were essential so students and testers could review detailed execution results and debug failures effectively.

The Approach & Solution

To overcome these challenges, I designed and implemented a robust automation solution:

  • Page Object Model (POM): The automation suite leveraged POM design patterns, isolating locators and actions for each page, ensuring maintainability when UI changes occurred.
  • Data-driven Testing: All test scenarios were data-driven, with input values managed in external Excel sheets. This approach reduced code repetition and simplified testing across various combinations.
  • Cross-browser Validation: Tests were executed on Chrome, Firefox, and Edge browsers using Selenium WebDriver and Grid, ensuring consistent user experiences.
  • Exception Handling & Logging: Built-in exception handling and custom logging were incorporated to capture failures, screenshots, and stack traces for faster root-cause analysis.
  • Reporting: Extent Reports was integrated for comprehensive HTML reports featuring step-level logs, screenshots, and detailed execution status.

Manual exploratory testing supplemented the automation to cover edge cases and usability aspects that required human judgment. Collaboration with other testers ensured thorough coverage of all business scenarios.

The Results

The project delivered excellent outcomes:

  • Achieved 95% automation coverage of core workflows, significantly reducing manual effort and increasing test consistency.
  • Reduced manual testing time by 80%, freeing resources for exploratory and usability testing.
  • Identified and reported critical functional defects early in the testing cycle, improving the app’s reliability for training purposes.
  • Produced professional, easy-to-read reports with screenshots and detailed logs, which were extremely valuable for students and testers referencing the project.
  • Contributed to maintaining a stable environment for learners worldwide, supporting the platform’s reputation as a go-to tool for automation practice.

Overall, the Adactin Hotel App project was a success, showcasing my ability to build scalable automation solutions, handle dynamic applications, and deliver clear, actionable insights through reporting.
